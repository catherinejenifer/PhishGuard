# PhishGuard: Phishing Email Detection & Awareness System

## 📌 Project Overview

PhishGuard is a cybersecurity project focused on detecting phishing emails using manual analysis techniques. This project simulates how a Security Operations Center (SOC) analyst investigates suspicious emails.

## 🎯 Objective

* Identify phishing indicators in emails
* Classify emails as Safe, Suspicious, or Phishing
* Understand common social engineering techniques

## 🛠️ Tools & Technologies

* Linux (Kali / Ubuntu)
* Manual Email Analysis
* GitHub

## 📂 Project Structure

* `samples/` → Contains email samples used for analysis
* `analysis/` → Contains detailed phishing analysis

## 🔍 Methodology

Each email was analyzed using the following steps:

* Sender verification (checking domain authenticity)
* Link inspection (identifying suspicious URLs)
* Content analysis (urgency, threats, tone)
* Risk classification

## 🚨 Key Findings

* Phishing emails often use spoofed domains (e.g., paypaI instead of paypal)
* Urgency and fear tactics are commonly used
* Fake links mimic legitimate websites

## 🛡️ Recommendations

* Always verify the sender’s email address
* Avoid clicking unknown or suspicious links
* Enable multi-factor authentication (MFA)
* Be cautious of urgent or threatening messages
